Technical: Version Control Concept 4/11/2015

What are the benefits of version control? Version control allows coder to keep track of different changes made to the code. The benefits is to allow users to easily go back to previous version of code if there's a mistake being made. Think of version control as a check point when playing a video game!

How does git help you keep track of changes? Git helps you git track of it by allowing you to add a modified file to stage, and determining if it's worth the change before commiting to it.

Why use Github to store your code? Think of github as an online place where you can access that file anywhere without your personal computer. It also allows online collaboration where different programmers can merge their code online and keep track of different changes.
